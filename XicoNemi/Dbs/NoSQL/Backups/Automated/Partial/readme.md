## 📌 Descripción

Esta sección contiene los scripts para realizar respaldos parciales automatizados de bases de datos NoSQL. Estos respaldos permiten a los administradores del sistema proteger información clave ante posibles cambios, fallos o incidencias en la operación de la plataforma.
