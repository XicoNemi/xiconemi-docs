## 📌 Descripción

La carpeta Poblate contiene una serie de scripts diseñados para poblar y administrar las bases de datos SQL del proyecto. Estos scripts permiten insertar datos de prueba o cargar información inicial, facilitando el desarrollo y las pruebas. Están orientados a la inserción de registros en tablas relacionales, asegurando la integridad de los datos y permitiendo simular escenarios reales para validar las funcionalidades implementadas.
