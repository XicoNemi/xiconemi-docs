## 📌 Descripción

(User-Defined Stored Procedures) se refiere a los procedimientos almacenados definidos por el usuario en una base de datos SQL. Son conjuntos de instrucciones SQL predefinidas que se almacenan y ejecutan en el servidor, permitiendo automatizar procesos, mejorar el rendimiento y optimizar la gestión de datos.
