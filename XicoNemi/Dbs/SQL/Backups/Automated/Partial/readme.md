## 📌 Descripción

Esta sección contiene los scripts para realizar respaldos parciales automatizados de bases de datos relacionales (SQL). Estos respaldos permiten a los administradores proteger información crítica, seleccionando tablas o datos específicos, ante posibles cambios, fallos o incidencias en la plataforma. Los scripts automatizados facilitan la ejecución periódica de respaldos, asegurando la integridad de los datos más importantes sin afectar el rendimiento del sistema.
