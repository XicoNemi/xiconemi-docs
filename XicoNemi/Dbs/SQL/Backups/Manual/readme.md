## 📌 Descripción

En esta carpeta se almacenan los scripts para realizar respaldos manuales de bases de datos SQL, tanto completos como parciales. Los respaldos completos guardan una copia de toda la base de datos, mientras que los parciales se enfocan en datos específicos, como tablas o esquemas seleccionados. Estos scripts permiten a los administradores realizar respaldos de manera manual, según las necesidades de la plataforma, asegurando la protección de los datos ante posibles fallos o incidencias.
