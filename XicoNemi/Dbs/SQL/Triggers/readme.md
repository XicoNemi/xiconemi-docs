## 📌 Descripción

Los triggers (disparadores) en SQL son procedimientos almacenados que se ejecutan automáticamente en respuesta a eventos específicos dentro de la base de datos, como inserciones, actualizaciones o eliminaciones de registros en una tabla. Se utilizan para automatizar procesos, mantener la integridad de los datos y gestionar auditorías sin intervención manual.
