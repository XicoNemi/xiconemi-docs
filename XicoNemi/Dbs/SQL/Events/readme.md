# 📂 Events 
## DBs/SQL/Events

![Events in SQL](https://img.shields.io/badge/Events_in_SQL-Database-orange)

## 📌 Descripción

El archivo events.sql define la estructura de la base de datos para la gestión de eventos y sus participantes, asegurando la correcta organización y relación entre las tablas. Incluye la creación de tablas para eventos, participantes y otras entidades relacionadas, el establecimiento de claves foráneas para mantener la integridad referencial y la inserción de datos de prueba para facilitar la validación y pruebas del sistema.
